Introduction

Windows Phone 8 SDK is now released. So, it's time for developers to start building WP8 apps. To start from, a hello world is always the best way to start. It shows the minimum files needed in your solution. It's always efficient and smart to start with simple code, understanding it well, then start developing more complex apps.

Building the Sample

First of all, you need to download Visual Studio 2012, and Windows Phone 8 SDK from 

http://go.microsoft.com/fwlink/?LinkID=257234&clcid=0x409.

After setting the tools needed, you just have to open the .sln file with Visual Studio 2012, and just press F5 to run the application and start the Emulator.

Description

As a welcome page, you will have this page showed during the run of the app.


 


After that, your app will start displaying in the Emulator, showing the Hello message.
